# Unified Mentor

## Introduction
Unified Mentor is a platform designed to connect mentors with mentees, facilitating knowledge sharing and personal growth. The goal of Unified Mentor is to create a seamless experience for users to find the right mentor or mentee for their needs.

## Features
- **Search Functionality:** Easily find mentors based on expertise, availability, and location.
- **Profile Creation:** Users can create detailed profiles that showcase their skills and experiences.
- **Messaging System:** Built-in chat feature to communicate between mentors and mentees.
- **Scheduling:** Integrated calendar for setting up meetings and sessions.

## Installation
To get started with Unified Mentor, follow these steps:
1. Clone the repository:
   ```
   git clone https://github.com/SayanMitra7/Unified-Mentor.git
   ```
2. Navigate to the project directory:
   ```
   cd Unified-Mentor
   ```
3. Install the required dependencies using:
   ```
   npm install
   ```

## Usage
Once the installation is complete, you can run the application using:
```bash
npm start
```
This will launch the application in your browser at `http://localhost:3000`.
